dataFormatting(data, dataType) → {Object|Array}

Function to format data according to the type of data

According the type of data the formatting is the following:

  • datatype = sessionUserActivity : session.day is converted into a number
  • datatype = sessionUserAverage : session.day is converted into a day of the week
  • datatype = userPerformance : convert fitness features from english to french language
  • datatype = todayScore : return data as an object containing the data for the score chart and the data of the score to display on the page

(async) getData(id, dataToRetrieve) → {Object|Number|Array}

Function which gets the data of the user either from the mock file or the API

Inside the function you can configurate the source of the data. You can choose if you want the data from the data mocked file or from the backend API. To make you choice, you just need to modify variable 'datamocked' which is a boolean, you can switch is value between values 'false' and 'true' to make your choice.
