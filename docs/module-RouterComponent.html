Module: RouterComponent

The component App() contains the router of the application SportSee. You can see in this component all the paths you can access inside the SportSee application.

Paths available:

  • /
  • /Home
  • /Profile/:userId
    Where userId is the id of the user you want to see the profil, for example: userId = 12 or userId = 18
  • *
    All other paths you can entering are leading to the error page
