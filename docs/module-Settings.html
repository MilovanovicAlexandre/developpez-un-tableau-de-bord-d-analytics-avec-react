Module: Settings

The fileapiCallsAndMockCall.jsx allows you to configurate the source of the data. You can choose if you want the data from the data mocked file or from the backend API. To make you choice, you just need to modify variable 'datamocked' which is a boolean, you can switch is value between values 'false' and 'true' to make your choice.
